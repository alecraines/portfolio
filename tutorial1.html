Lava Zones and Shelters in Hawaii

Project Description

This map was created in ArcGIS Online using publicly available datasets. Put together I created an illustration of the hazardous zones in Hawaii. Paired with the highways and emergency shelters in Hawaii, it allows for planning for residents to locate their nearest shelter in the case of an emergency.

Project Details

  • Map made in ArcGIS Online
  • Used USGS Lava Zones Datasets
  • Point data including the emergency shelters and volcanoes were added
