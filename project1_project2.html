Tour of Seattle with Google My Maps

Project Description

Using Google Maps, I made a Tour of Seattle. After finding 12 locations and gathering their coordinates and some information about the place, it was exported as a CSV into Google My Maps, then a KMZ file was used and then put into a website to be used with the Google Maps API.

Project Details

  • Utilised the Google Maps API to embed
  • Preselected sites in Google Sheets, then imported to My Maps
  • Each point and area has associated information with location
  • Included driving paths to each location
